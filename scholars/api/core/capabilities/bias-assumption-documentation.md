# bias-assumption-documentation

# Bias and assumption documentation for every statistic

**PI directive (this run):** all stats need documented bias and assumptions.

**Rule:** Each reported statistic must be accompanied by: (a) the denominator and inclusion definition, (b) known selection bias (e.g. cohort is amyloidosis-selected, synthetic data not clinically valid), (c) confounders that could distort it, and (d) the assumption that makes the number meaningful. Attach these inline where the number is stated, not only in a limitations section.
