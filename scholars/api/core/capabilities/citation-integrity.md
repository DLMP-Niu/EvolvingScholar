# Citation integrity

**PI directive (this run):** citations were present but links/publication IDs were not usable.

**Rule:** Every literature claim must carry a resolvable identifier — a DOI, a PMID, or a full URL that actually resolves — plus the specific claim it supports. Do not cite by journal name + year alone. Before finalizing, verify each identifier resolves; if it cannot be verified, mark the claim as unverified rather than citing a dead/guessed link.
