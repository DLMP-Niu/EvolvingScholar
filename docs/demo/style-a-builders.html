the bet

Most "self-improving" agents
just append to the prompt.

We ruled that out on purpose. Cross-project growth happens only through updates to external artifacts — concept model, method repertoire, knowledge base, guardrails. Working context stays inside a run; it never persists as a bigger prompt between runs.

// removes the context-length confound · keeps "the agent evolved" = a named diff

it caught something real

A data-quality bug, unprompted.

On the TTR/ATTR cohort, the scholar flagged 41 amyloidosis patients coded under legacy ICD-9 277.3x instead of E85.x — so a naïve E85 filter silently under-counts the cohort. It then wrote that lesson back as a reusable capability.
